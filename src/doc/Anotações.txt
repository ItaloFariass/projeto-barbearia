*TABELAS*

<table> -> Tag para inciar uma tabela como <ul>, <form> ...

<tr> -> Table ROW - são as linhas das tabelas 

<td> -> Table DATA - são as culunas / celulas das tabela 


*A tabela pode ser dividida como o html, por head, body e footer* 


<thead> -> cabeçalho da tabela, as celulas (<td>) são marcadas como <th>

<tbody> -> Corpo da tabela, as tags não mudam

<tfoot> -> O rodapé da tabela
